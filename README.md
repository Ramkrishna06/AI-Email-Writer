# AI Email Writer

AI Email Writer is a full-stack AI-powered web application that helps users generate professional, well-structured email replies instantly. It leverages modern frontend technologies with a robust backend and integrates Google Gemini AI to produce context-aware responses.

This project is designed to improve productivity, save time, and showcase real-world AI + full-stack development skills.
